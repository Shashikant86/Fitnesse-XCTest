# Fitnesse-XCTest
Demo of Running Fitness Tests with XCTest. 

# Quick Start

Clone the repo and execute Fitnesee Tests as XCTest

             $ git clone https://github.com/Shashikant86/Fitnesse-XCTest
             $ cd Fitnesse-XCTest
             $ bundle install 
             $ bundle exec fastlane xctest_fitnesse
